## Notebook — Everyday Design System

### Key decisions
- **Site name**: Everyday Design System
- **Public access**: All sections public (no auth gating for now)
- **Fonts**: Work Sans (primary), Geist Mono (mono)
- **Deploy**: Vercel, domain `everyday.co`

### Source assets
- Logos/emblems live in `logo_files/`:
	- `logo_files/prop_*.svg` (Everyday Properties variants)
	- `logo_files/home_*.svg` (Everyday Homebuyers variants)
	- `logo_files/emblem_*.svg` (emblem variants)
	- `logo_files/Main.png` (Homebuyers lockup, green/navy)

### eBay Playbook patterns we’re mirroring (baseline)
- Top nav: Get started / Brand strategy / Foundations / Design system / Expressions / Resources / Articles
- Section landing pages: title + short description + Grid/List toggle + curated list of subpages
- Tools & resources: filter pills via query params + Grid/List toggle + resource list
- Get started subnav: Overview / FAQ / System definitions (left sidebar)
